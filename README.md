# Projeto Matriz

Este projeto tem como finalidade a implementação de um sistema que permitirá gerenciar a criação de matrizes curriculares.

# Tecnologias usadas

<ul>
    <li>Python</li>
    <li>Django</li>
    <li>Postgresql</li>
</ul>
