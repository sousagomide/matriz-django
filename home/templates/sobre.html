Sobre

Sistema de Gerenciamento Matricial

O Sistema de Gerenciamento Matricial (SGM) foi desenvolvido com intuito de aperfeiçoar as técnicas de programação utilizando Python e Django. Esse sistema funciona sob uma arquitetura MVT e utiliza o banco de dados PostgreSQL. Este projeto é público e seu código pode ser baixado e utilizado por qualquer desenvolvedor.

O template do projeto foi utilizado a partir de um tema gratuito que pode ser obtido no site Start Bootstrap
